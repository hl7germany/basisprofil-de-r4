### ICD-10 GM

Bei der Kodierung mittels [ICD-10-GM](https://www.bfarm.de/DE/Kodiersysteme/Klassifikationen/ICD/ICD-10-GM/_node.html) muss mindestens ein verwendetes Coding den Anforderungen der ICD-10-GM-Kodierung entsprechen. Das entsprechende ICD-10-GM-ValueSet ist auf dem zentralen Terminologieserver des deutschen Gesundheitswesens unter [terminologien.bfarm.de](https://terminologien.bfarm.de/) verfügbar und kann für ein Binding im jeweiligen Use-Case-Profil sowie zur terminologischen Validierung herangezogen werden.

Die Angabe der ICD-Version (z.B."2019"), aus der ein Code stammt, ist verpflichtend, das ICD-GM nicht versions-stabil ist, d.h. Codes können zwischen den unterschiedlichen Jahresangaben in der Bedeutung wechseln.
