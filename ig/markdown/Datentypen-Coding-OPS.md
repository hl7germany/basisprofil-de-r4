### OPS

Bei der Kodierung mittels [OPS](https://www.bfarm.de/DE/Kodiersysteme/Klassifikationen/OPS-ICHI/OPS/_node.html) muss mindestens ein verwendetes Coding den Anforderungen der OPS-Kodierung entsprechen. Das entsprechende OPS-ValueSet ist auf dem zentralen Terminologieserver des deutschen Gesundheitswesens unter [terminologien.bfarm.de](https://terminologien.bfarm.de/) verfügbar und kann für ein Binding im jeweiligen Use-Case-Profil sowie zur terminologischen Validierung herangezogen werden.

Die Angabe der OPS-Version (z.B."2019"), aus der ein Code stammt, ist verpflichtend, da OPS nicht versions-stabil ist, d.h. Codes können zwischen den unterschiedlichen Jahrensangaben in der Bedeutung wechseln.
