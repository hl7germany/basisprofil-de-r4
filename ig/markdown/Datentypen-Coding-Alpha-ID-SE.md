### Alpha-ID-SE

Bei der Kodierung per [Alpha-ID-SE](https://www.bfarm.de/DE/Kodiersysteme/Terminologien/Alpha-ID-SE/_node.html) muss das verwendete Coding-Element mit mindestens einem Coding gefüllt sein, das den Anforderungen der Alpha-ID-SE-Kodierung genügt. Hierzu sollte beim Einbinden des Coding-Profils in das entsprechende Use-Case-Profil bei Bedarf ein projektspezifisches Binding oder ein geeigneter Terminologiedienst verwendet werden.

Die Angabe der Alpha-ID-SE-Version (z.B."2019"), aus der ein Code stammt, ist verpflichtend, da die Alpha-ID-SE nicht versions-stabil ist, d.h. Codes können zwischen den unterschiedlichen Jahrensangaben in der Bedeutung wechseln.
